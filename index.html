Rtype-parser

Parser and editor for Rtype type definitions.

Download .zip Download .tar.gz View on GitHub

This repository contains a parser and an editor for Rtype type definitions. The parser is generated by ANTLR4 based on the grammar Rtype.g4. The editor is based on ACE editor, but uses the ANTLR4 generated lexer for syntax highlighting (see antlr4-ace-ext, my-mode.js and rtype-token-type-mapping.js) and the ANTLR4 generated parser to provide error annotations (see my-worker.js).

This project is in early stage. If you like to join the discussion
